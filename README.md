# 📢 Subscribe Popup

Um popup de inscrição em canal do YouTube, com avatar personalizado, nome do canal e botão de **Subscribe** — tudo sobre um fundo animado com efeito de raios coloridos.

---

## 📸 Preview

> Popup centralizado com avatar "CL", nome **Júlia**, texto "Subscribe Our YouTube Channel" e botão azul de Subscribe, sobre fundo com efeito de luz irradiante colorida.

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**

---

## 📁 Estrutura do Projeto

```
subscribe-popup/
├── index.html
├── style.css
```

---

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/subscribe-popup.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd subscribe-popup
   ```

3. Abra o `index.html` no navegador, **ou** utilize o **Live Server** do VS Code:
   - Instale a extensão **Live Server**
   - Clique com o botão direito em `index.html` → **Open with Live Server**
   - Acesse em: `http://127.0.0.1:5500/index.html`

---

## 🎯 Funcionalidades

- Popup de inscrição exibido sobre a página
- Avatar circular com iniciais do canal (**CL**)
- Nome do canal: **Júlia**
- Texto de chamada: *Subscribe Our YouTube Channel*
- Botão **Subscribe** estilizado em azul
- Botão de fechar (✕) no canto superior do popup
- Fundo animado com efeito de raios de luz coloridos

---

## 🎨 Design

- Fundo: textura animada com raios de cores suaves (branco, rosa, azul claro)
- Popup com fundo semitransparente/branco e bordas arredondadas
- Avatar roxo/lilás com iniciais em branco
- Botão Subscribe em azul ciano com texto branco
